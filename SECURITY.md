# Security Policy

## Supported versions

`0.1.x` is a Fedora 44 x86_64 technical preview. Keep DNF5 installed and available for recovery.
No other release or distribution is currently supported.

## Reporting a vulnerability

Use GitHub private vulnerability reporting at
<https://github.com/GPLaider/dnfast/security/advisories/new>. Do not open a public issue for a
suspected trust-bypass, signature-verification, privilege-boundary, RPMDB-integrity, or transaction
recovery defect.

Include the exact version or commit, Fedora version, command, relevant unmodified logs, and a minimal
reproducer. Remove credentials and private repository URLs, but do not rewrite other evidence.
